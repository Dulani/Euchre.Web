# Euchre.Web (Vanilla JS Edition)

A web version of the classic card game Euchre, rebuilt in Vanilla JavaScript for easy static hosting.

## Features
- Full Euchre game logic including bidding rounds and Bowers.
- AI players for North, East, and West positions.
- Responsive design with a collapsible sidebar and card table.
- Game Log and scoring tracking.
- Interactive UI with highlights for playable cards.

## Hosting
This project is designed to be hosted as a static site, for example on GitHub Pages at `dulani.github.io/Euchre.Web`.

## Getting Started
Since this project uses Vanilla JS and CDN-hosted libraries (Tailwind CSS), there is no build step or installation required.

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Or serve using a simple HTTP server: `npx serve .`
